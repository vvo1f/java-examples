# Compilers 

NOTE: To work with the practices and examples, you need to download, install and configure Jflex, Java, and Cup in your operating system.
NOTE: It's highly recommended to have jdk-13.0.2 on your system to execute the examples.


# To execute files

### $ jflex <file-name>.jflex

### $ javac <class-name>.java

### $ java <class-name> <file-name-1> <file-name-2> ...
